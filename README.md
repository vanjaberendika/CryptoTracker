# CryptoTracker (CurrencyTracker)

Real-time currency tracking Android app built with **Kotlin**.

Track **3 fiat currencies** and **5 crypto assets**, set a **default fiat currency**, and manage your **favorites list** with persistent storage. Includes **Light/Dark theme** and a clean multi-tab layout.
